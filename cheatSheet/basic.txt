GREP
====
cat txt.txt | grep -n "waterloo"
parses txt.txt file for the word "waterloo" and provides line numbers

Deletion
====
rm -R /path		-for directories with stuff
rm /path		-for single files
rmdir /path		-for empty directories

TAR
====
Extract/install
tar
	tar xf /path
tar.xz / tar.zst
	-install directly via
    pacman -U /path
tar.gz
	tar -xzf /path
tar.7z
	-requires 7z package
    7z e /path
Misc
	-see whats inside tarball
	tar tf /path
	-see what will be installed
    pacman -Qlp /path
INSTALL
=======

makepkg -si
patch < something.diff
pacman -S something
	-install from repo
pacman -Q sonething 
	-info about package on comp
pacman -U something
	-install from local
pacman -Ss something
	-seach repos
