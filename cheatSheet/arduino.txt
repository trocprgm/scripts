COMPILE AND UPLOAD
##################

list boards
arduino-cli board list

compile
arduino-cli compile --fqbn arduino:avr:uno Blink

upload
arduino-cli upload -p /dev/ttyACM0 --fqbn arduino:avr:uno Blink

create compile/upload scripts
echo "arduino-cli compile --fqbn arduino:avr:uno Blink" >> compile
echo "arduino-cli upload -p /dev/ttyACM0 --fqbn arduino:avr:uno Blink" >> upload
add #!/bin/bash and chmod +x 

SERIAL MONITOR
##############

picocom -b 9600 /dev/ttyACM0

UPDATE CORES
############

check for installed cores
arduino-cli core list

update cores from arduino-cli.yaml config
arduino-cli core update-index

install new core (after adding it to arduino-cli.yaml)
arduino-cli core install esp32:esp32 

FIND FBQN
#########

arduino-cli board listall esp32
arduino-cli board listall esp32cam
                            etc...
OR
upload via IDE and read first line of console output for fbqn
